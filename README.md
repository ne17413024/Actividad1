# Tareas Desarrollo Full Stack

Este repositorio contiene las tareas y actividades prácticas del curso de Desarrollo Full Stack del cuarto semestre en la Universidad Tecmilenio.

## Objetivo

El propósito de este repositorio es almacenar y organizar las actividades realizadas durante el curso, demostrando el aprendizaje progresivo en el desarrollo de aplicaciones web full stack.

## Resumen de Actividades

- **Actividad 1**: Creación de una página web estática básica utilizando HTML y CSS. Incluye un diseño simple y funcional para una empresa tecnológica.
- **Actividad 2**: Desarrollo de una interfaz web más avanzada con estilos personalizados, layout responsivo y elementos de navegación.

Cada actividad está contenida en su propia carpeta, con archivos HTML, CSS y documentación adicional.

## Tecnologías Utilizadas

- HTML5
- CSS3

## Cómo Ejecutar

Para visualizar cada actividad, abre el archivo `index.html` correspondiente en un navegador web moderno.
